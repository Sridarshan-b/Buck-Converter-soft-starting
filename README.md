# \# DC-DC Buck Converter with Soft Starting

# 

# \## Overview

# A hardware-implemented DC-DC Buck Converter with a soft-starting feature

# using TI C2000 (F2806x) microcontroller to limit inrush current during

# startup. Designed and tested as part of coursework at SASTRA Deemed University.

# 

# \## Key Specifications

# \- Input Voltage: 12 V

# \- Output Voltage: 5 V

# \- Switching Frequency: 150 kHz

# \- Soft-start duration: 75 ms

#
